# Maruti Portfolio

This is a single-page static portfolio ready for Vercel.

## Deploy on Vercel without coding
1. Create a folder with this `index.html` file.
2. Zip the folder if needed.
3. In Vercel, create a new project and import/upload the folder or connect a GitHub repo.
4. Deploy.
5. Add a custom domain later if you want.

## What to edit
- Replace placeholder email and LinkedIn
- Add exact numbers in case studies
- Update the title if you want Product Manager positioning
